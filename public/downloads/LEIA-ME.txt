DESCRIÇÃO:
Este software converte arquivos NetCDF (.nc) para CSV.
Ideal para processar arquivos muito grandes (3GB, 4GB ou mais).

══════════════════════════════════════════════════════════════════

REQUISITOS:

- Windows 10 ou 11

Não precisa instalar Python nem bibliotecas!
Basta baixar e executar.

══════════════════════════════════════════════════════════════════

COMO USAR:

1. Baixe o arquivo "ConversorNetCDF-DefesaCivil.exe"

2. Dê duplo clique para abrir o programa

3. Selecione os arquivos .nc (você pode escolher 1 ou vários)

4. Escolha a pasta de destino para os CSVs

5. Opções extras disponíveis:
   * Unificar todos em um único arquivo CSV
   * Calcular média anual automaticamente (Gera um CSV de resumo)

6. Clique em "CONVERTER PARA CSV"

══════════════════════════════════════════════════════════════════

FUNCIONALIDADES:

- Processa vários arquivos de uma vez (Lote)
- Unifica vários arquivos em um único CSV
- Cálculo automático de média anual
- Sem limite de tamanho (3GB, 10GB+)
- Funciona offline
- Interface gráfica moderna

══════════════════════════════════════════════════════════════════

SOLUÇÃO DE PROBLEMAS:

O Windows pode exibir um aviso de segurança:
   → Clique em "Mais informações" e depois em "Executar assim mesmo"
   → Isso acontece porque o programa não tem assinatura digital

Erro: "MemoryError"
   → Feche outros programas para liberar memória
   → Aumente a memória virtual do Windows
